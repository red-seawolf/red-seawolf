⚙ Live backend (optional)

Shopee and Lazada block scraping from inside a browser page, so live merged results need the small self-hostable server from this project (cd price-compare && npm install && npm start — see the README). Run it on any machine you control, then connect it here. Without a backend, the Demo source still runs fully in your browser and Shopee/Lazada open as direct marketplace links.

Aggregates public listings for comparison. Price per unit is derived from each product title and is a best-effort estimate. Works offline after the first load and can be installed to your home screen.
